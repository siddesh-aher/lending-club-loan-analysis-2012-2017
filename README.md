# Lending Club Loan Analysis (2012–2017)

This repository contains an end-to-end analysis of Lending Club peer-to-peer loan data from 2012 to 2017 using R.

## Project Overview
The analysis focuses on:
- Geographic distribution of loans across U.S. states
- Loan grades and associated interest rates
- Borrower income and employment characteristics
- Time-series trends from 2012–2017

## Tools & Technologies
- R
- tidyverse
- ggplot2
- usmap
- janitor
- lubridate

## Files
- Lending_Club_Analysis.R – Main analysis script
- Final-Case-Analysis.pdf – Final written report
- Final Case Analysis - Case Study.docx – Editable report version
